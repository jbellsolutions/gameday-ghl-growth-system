# Game Day Claude Code Training Proposal

## Objective
Train Game Day office staff to use the Claude Code ecosystem in role-specific workflows, then place an AI Integrator to run automation daily and prove measurable outcomes at Tony's two locations before franchise rollout.

## Offer Structure

### Tier 1: Certification Seats (Level 1)
- 5 seats for core team members (Front Desk, Clinic Ops, Marketing, Owner/GM, Integrator Candidate)
- Access to Claude Code Ecosystem Certification curriculum
- Completion target: 14 days

### Tier 2: Role-Based Implementation Training
- 4 live coaching calls (weekly)
- Role-specific SOPs and prompt packs
- 1:1 office-hours support seats for blockers
- Customized workflows mapped to Game Day operations

### Tier 3: AI Integrator Placement + Coaching
- AI Integrator placement: `$1,000` one-time
- Ongoing support/coaching: `$300/month`
- Weekly KPI accountability + optimization reviews
- Integrator runs GHL workflows + content/ops automations day-to-day

## Recommended Package
- Setup + training sprint: `$11,000` one-time
- Ongoing support: `$1,050/month`

## Role Tracks (Game Day)

| Role | What They Learn | Immediate Outcome |
|---|---|---|
| Front Desk | Lead intake prompts, objection handling, appointment rescue scripts | Faster booking and fewer dropped leads |
| Clinic Ops | Pipeline QA, no-show recovery checks, daily KPI loops | Consistent workflow performance |
| Marketing | Expert Series repurposing, campaign scheduling, newsletter cadence | Steady multi-channel content output |
| Owner / GM | Dashboard analysis prompts, location comparison, decision memos | Faster leadership decisions |
| AI Integrator | End-to-end system operation, escalation rules, weekly optimization | One accountable operator running the stack |

## Prove-Then-Franchise Rollout

### Phase 1 (Weeks 1-2): Internal Enablement
- Enroll 5 team members into Level 1 + role tracks
- Run first coaching sprint
- Publish role SOPs and prompt libraries

### Phase 2 (Weeks 3-6): Tony Location Proof
- Deploy AI Integrator at Tony's locations
- Run weekly KPI cadence:
  - Reactivations booked
  - Speed-to-lead response time
  - No-show recovery rate
  - Reviews generated

### Phase 3 (Weeks 7-12): Case Study Packaging
- Build a "Game Day Certified AI Clinic" proof package
- Capture before/after KPI deltas and playbooks
- Prepare franchise intro deck and implementation offer

### Phase 4 (Month 4+): Franchise Expansion
- Offer program to additional locations
- Standardize onboarding (seat bundle + integrator + KPI dashboard)

## Why This Works
- Role-based training removes generic "AI education" waste.
- AI Integrator creates daily execution consistency.
- Coaching calls reduce adoption friction.
- Proof at Tony's clinics lowers risk for franchise rollout.

## Success Criteria (First 90 Days)
- 5/5 core seats certified
- AI Integrator active and operating weekly cadence
- Speed-to-lead under 5 minutes
- Reactivation campaign live and measurable
- Publish one internal case study for franchise conversations

## Decision Summary
Yes, take this on.
- Start with internal staff training and one integrator at Tony's locations.
- Prove outcomes in 90 days.
- Use that proof to sell standardized adoption across franchise locations.
